# API 网关开发任务

本任务清单承接 [PRD](../../prd.md)、[需求规格](requirements.md) 和 [技术设计](design.md)。任务编号采用 `TASK-里程碑序号xx`，每项任务需要能追溯到 `FR`、`NFR` 或 `DD`。

任务状态：

- `[ ]` 未开始
- `[~]` 进行中
- `[x]` 已完成

## 当前最小 MVP 调整

根据当前交付目标，先完成“最小 ThinkPHP API 网关骨架”，范围包括路由、伪静态、访问日志、统一响应、CORS 和简单上游代理。应用管理、数据库模型、签名鉴权、限流和管理 API 暂后。

| [x] | MVP-002 | 配置 `/gateway/*` 简单代理路由。 | FR-007、FR-012 |
| [x] | MVP-003 | 增加伪静态入口配置。 | NFR-004 |

| [x] | MVP-005 | 增加浏览器跨域 CORS 和 OPTIONS 预检响应。 | NFR-001 |
